## 🧩 About the Project

This project is a **curated directory of AI tools** designed for real‑world work. It helps users quickly discover reliable AI products across multiple categories, mark favorites, and stay up‑to‑date with new additions.

The platform currently lists **128+ AI tools** across **8 categories**, with regular updates and community contributions.

> 💡 **Purpose:** Cut through the noise of AI tooling by providing a cleaner, more focused directory — built for founders, marketers, developers, designers, and creators alike.

---

## ✨ Key Features

- **Category‑based browsing** – Tools are grouped into writing, image generation, coding, design, video/audio, research, productivity, and marketing.
- **Favorites system** – Mark and save your preferred tools for quick access.
- **Random discovery** – Explore new tools with a random pick feature.
- **Regular updates** – New tools are added frequently (e.g., Galileo AI, AdCreative.ai, Ocoya, Motion, Relume AI, Bolt.new, etc.).
- **Live stats** – Dashboard shows total tools listed, recent additions, and last update date.
- **Community contribution** – Users can suggest missing AI tools or report broken links.
